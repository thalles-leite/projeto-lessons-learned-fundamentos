MEU APRENDIZADO

Durante a primeira semana de curso de Desenvolvimento Web na Trybe é possível listar inúmeros aprendizados, tanto na área de Hard Skills quanto na área de Soft Skills. O aprendizado já começa quando é necessário utilizar o Linux, SO amplamente utilizado no mundo Dev. Após a instalação do mesmo foi possível entender o significado dos termos Unix e Bash, a importância da utilização de um SO (Sistema Operacional) baseado no Unix e os principais comandos Bash.Foi possível compreender também sobre o processo de versionamento de código utilizando o Git e o quanto o mesmo nos possibilita uma boa organização do projeto, principalmente quando envolve uma equipe. E para a hospedagem dos códigos com controle de versão, aprendemos utilizar as principais ferramentas do GitHub. Em seguida, aprendemos os fundamentos da linguagem de marcação HTML e a linguagem de folha de estilos CSS.
Dos valiosos aprendizados que obtivemos essa semana de Hard Skills, posso citar o quanto é importante o desenvolvimento da Autoliderança, o poder planejamento, o aprimoramento da comunicação utilizando assertividade e as técnicas para bons feedbacks.
